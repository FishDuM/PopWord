# PopWord 弹词 - 单词学习浏览器插件

PopWord 弹词是一个浏览器插件，通过点击网页任意位置显示单词及读音，帮助用户在浏览网页的同时学习英语单词。

## 功能特性

- **点击显示单词**：在网页任意位置点击鼠标左键，显示下一个单词；点击鼠标右键，显示上一个单词
- **音频发音**：自动播放单词的发音，帮助用户学习正确的发音
- **词库选择**：支持从多个词库中选择，默认使用 CET4 词汇
- **设置中心**：提供直观的设置界面，可调整显示选项和音频设置
- **音频 API 自定义**：支持手动输入指定的音频 API
- **缓存管理**：智能缓存音频，提高播放速度
- **浮动效果**：单词以浮动效果显示，慢慢上升并淡出
- **记忆功能**：自动记录单词学习历史，按日期分类
- **监控面板**：查看单词学习历史，按日期分类，支持点击单词播放音频
- **跨页面单词序列**：所有页面使用同一个单词序列，确保学习的连续性

## 安装方法

1. 克隆或下载本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目目录，完成安装

## 使用说明

1. 安装完成后，浏览器右上角会出现 PopWord 弹词 插件图标
2. 点击插件图标打开设置面板
3. 调整各项设置，如是否同时显示单词和中文意思、是否播放音频等
4. 在任意网页上点击鼠标左键，即可看到下一个单词以浮动效果显示
5. 点击鼠标右键，即可看到上一个单词以浮动效果显示
6. 如果设置为分开显示单词和中文意思：
   - 左键第一下显示英文单词
   - 左键第二下显示中文意思
   - 右键第一下显示上一个英文单词
   - 右键第二下显示上一个的中文意思
7. 点击"监控面板"按钮，查看单词学习历史
8. 在监控面板中，点击日期行可以收起/展开该日期下的单词
9. 点击监控面板中的单词可以播放对应的音频
10. 点击"删除缓存"按钮可以清除所有缓存的音频和学习历史

## 技术栈

- **前端**：HTML, CSS, JavaScript
- **浏览器扩展**：Chrome Extension Manifest V3
- **存储**：Chrome Storage API
- **音频**：Web Audio API, Youdao Dictionary API

## 核心功能

### 单词显示

- 点击网页任意位置，显示单词及其中文意思
- 支持两种显示模式：同时显示单词和中文意思，或分开显示
- 单词以浮动效果显示，慢慢上升并淡出
- 鼠标左键显示下一个单词，鼠标右键显示上一个单词

### 音频播放

- 自动播放单词的发音
- 使用 Youdao Dictionary API 获取音频
- 支持自定义音频 API
- 智能缓存音频，提高播放速度
- 监控面板中点击单词可播放音频

### 词库管理

- 支持从多个词库中选择
- 自动扫描词库文件夹中的 JSON 文件
- 默认使用 CET4 词汇词库
- 支持重置词库，从第一个单词开始

### 缓存管理

- 初始化时缓存 10 个单词的音频
- 每使用 5 个单词后，清除缓存并重新缓存
- 最大缓存 15 个单词的音频
- 实时显示当前缓存大小
- 支持清除所有缓存的音频和学习历史

### 记忆功能

- 自动记录单词学习历史，包括单词、词性、中文意思和学习日期
- 按日期分类存储学习历史
- 支持查看历史记录，按日期排序
- 跨页面保持单词学习序列

### 监控面板

- 按日期分类显示单词学习历史
- 支持点击日期行收起/展开该日期下的单词
- 支持点击单词播放对应的音频
- 显示单词的词性和中文意思

## 词库格式

词库文件使用 JSON 格式，每个单词包含以下信息：

```json
{
  "word": "abruptly",
  "translations": [
    {
      "translation": "突然地",
      "type": "adv"
    }
  ],
  "phrases": [
    {
      "phrase": "absorb in",
      "translation": "集中精力做某事；全神贯注于"
    }
  ]
}
```

## 开发指南

1. 修改 `content.js` 文件以调整核心功能
2. 修改 `popup.html` 和 `popup.js` 文件以调整设置面板和监控面板
3. 在 `world` 文件夹中添加新的词库文件
4. 修改 `manifest.json` 文件以调整插件配置
